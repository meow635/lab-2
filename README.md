# Lab 2: Parallel Matrix Multiplication (OpenMP)

## Описание
Программа реализует параллельное перемножение квадратных матриц с использованием OpenMP.

## Используемые технологии
- C++
- OpenMP
- Python (для построения графиков)

## Параллелизация
Параллелизация выполнена по внешнему циклу (по строкам матрицы), что исключает гонки данных.

## Эксперименты

Проведены вычисления для:

Размеры матриц:
- 200, 400, 800, 1200, 1600, 2000

Количество потоков:
- 1, 2, 4, 8
